如何用css制作下拉菜单

2014-08-21

制作思路：第一步：静态网页的制作；第二步：动态特效的实现；第三步浏览器的兼容

静态网页的制作

大体属性：

1、标签<ul><li>...</li></ul>项目制作

2、标签<a>链接

3、float浮动

4、display：block；属性

5、position属性规定元素的定位

   
<html xmlns="http://www.w3.org/1999/xhtml">
<head>
<meta http-equiv="Content-Type" content="text/html; charset=utf-8" />
<title>无标题文档</title>
<style type="text/css">
*{ margin:0px; padding:0px;} /*清楚默认样式*/
 #menu{ background-color:#eee; width:600px; height:40px; margin:0 auto;} /*定义一个区域，背景颜色为灰色，宽度为600像素，高度为40像素，水平居中显示*/
 ul{ list-style:none;}/*清楚列表的项目符号*/
 ul li{ line-height:40px; text-align:center; position:relative; float:left; } /*定义字体行高为40像素，居中显示，相对定位，左浮动*/
 a{ text-decoration:none; color:#000; display:block; width:90px;} /*清除链接文字下划线，设置字体颜色为黑色，定义块元素，宽度为90像素*/
 a:hover{ color:#FFF; background-color:#666;} /*设置鼠标经过时的字体颜色，和背景颜色*/
 ul li ul li{ float:none;margin-top:2px; background-color:#eee; } /*清楚下拉菜单的浮动属性，设置上边距为10像素，背景颜色为浅黑色*/
 ul li ul{width:90px; position:absolute;display:none;} /设置下拉菜单的宽带为90像素，定位方式为绝对定位，隐藏块元素/
 ul li:hover ul{ display:block;} /*设置鼠标经过时显示快元素*/
</style>
</head>
<body>
<div id="menu">
<ul>
  <li><a href="#">首页</a></li>
  <li><a href="#">课程大厅</a>
    <ul>
      <li><a href="#">JavaScript</a></li>
      <li><a href="#">jQuery</a></li>
    </ul>
  </li>
  <li><a href="#">学习中心</a>
    <ul>
      <li><a href="#">视频学习</a></li>
      <li><a href="#">案例学习</a></li>
      <li><a href="#">交流平台</a></li>
    </ul>
  </li>
  <li><a href="#">经典案例</a></li>
  <li><a href="#">关于我们</a></li>
</ul>
</div>
</body>
</html>
效果图：

总结分析： 首先清除默认样式，通过div设置宽高，背景，形成一块独立区域，设置margin属性，让这块区域居中显示，在div中增加一个无须列表，清楚列表项目符号 ，通过浮动属性，让列表字体在这个区域横向显示，设置每个列表项的宽度（不然相距很近不美观），再通过定义行高，设置字体对齐方式 让字体水平居中显示，在<li></li>嵌套列表的方法来制作下拉菜单，让后清除float属性让他竖向显示 通过<ul>绝对定位,和父元素<li>的相对定位，让父元素<li>不随子元素的长度变化而变化。通过设置display:none;让下拉菜单隐藏，在通过hover属性设置鼠标经过显示下拉拉菜单，字体颜色和背景颜色的变化。
